## Requirements

- Python 3
- Test case file

## Usage

- python3 main.py \<path of the test case file>

## Examples

```
python3 main.py test_cases/input1.txt 
Terminal: +, *, (, ), id
Non terminal: E, T, EPrime, F, TPrime
```

```
python3 main.py test_cases/input2.txt
Terminal: +, *, id, (, )
Non terminal: E, T, F
```
